# 安装&使用

::: tip 提示

@wocwin/t-ui 是基于 vue2 + Element-ui  二次封装的基础组件

:::

## Install

```js
npm install @wocwin/t-ui -S
&
yarn add @wocwin/t-ui -S
```

## Usage

```js
// main.js
import Tui from "@wocwin/t-ui"

Vue.use(Tui)
```
